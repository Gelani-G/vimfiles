### How to install ###

* Clone repo 

```
git clone https://github.com/evserykh/vimfiles.git ~/.vim
```

* Clone [vundle](https://github.com/gmarik/vundle)

```
git clone https://github.com/gmarik/vundle.git ~/.vim/bundle/vundle
```

* Add to your .vimrc

```
source ~/.vim/vimrc
```

* Run Vim and execute

```
:BundleInstall
```
